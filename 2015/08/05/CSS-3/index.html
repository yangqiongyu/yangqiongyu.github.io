选择器

与jQuery中大部分的选择器兼容

  • 属性选择器
    • E[attr] input[type=”text”]
    • E[attr~=]表示的一个单独的属性值，这个属性值是以空格分隔的
    • E[attr|=val]表示的要么一个单独的属性值，这个属性值是以“-”分隔的
    • E[attr*=val]表示的是只要val里包含某个字符并且在“任意”位置
    • E[attr^=val]表示的属性值里包含val字符并且在开始位置
    • E[attr$=val]表示的属性值里包含val字符并且在“结束”位置，独立的也可以匹配到
  • 伪类选择器
    • dd:first-child 相对于父级做参考，所有子元素的第一个子元素，并且位置要对应
    • dd:first-of-type 相对于父级做参考，“特定类型”（）的第一个子元素
    • dd:nth-of-type(2) 相对于父级做参考，“特定类型”（）的第二个子元素
    • span:last-child：选中最后一列
    • E:only-child 相对于父元素做参考，表示父元素只有一个子元素的E
    • E:nth-child(n)
  • 伪元素选择器
    • E:selection 可该表选中文本的样式
    • E::placeholder 可改变placeholder默认样式
    • first-letter
    • ：：after E:after 、E:before在旧版本里是伪类，在新版本里是伪元素，新版本下E:after、E:before会被自动识别为E::after、E::before，按伪元素来对待
    • ：：before

文本

      • -webkit-box-orient:vertical;

边框圆角

  • border-radius:宽高的一半 就是个正圆
  • 拱形：border-radius：90px 90px 0 0
  • 半圆：border-radius：90px（和高度一致） 90px 0 0
  • 左上角：border-radius：90px 0 0 0
  • 四分之一圆：高度为90px border-radius：90px 0 0 0;
  • 横着的椭圆：高度为90px，宽度为180px border-radius：50%；或者border-radius：90px/45px;长轴设置为90，短轴设置为45
  • 竖直的椭圆：宽90 border-radius：45px/90px
  • 半个横着的椭圆：height：45px border-radius：90px 90px 0 0/45px 45px 0 0 四个角的横轴/纵轴
  • 半个竖着的椭圆：width：45px border-radius：45px 0 0 45px/90px 0 0 90px
  • 四分之一竖着的椭圆：width：45px height: 90px border-radius:45px 0 0 0/90px 0 0 0
  • 阴阳图，宽度180 border-top-width：90px border-bottom-width:90

    • border-radius：50%(水平时) 50%（垂直时）；
    • 百分比是按横竖两个对应的方向的长度进行计算的

边框阴影

  • box-shadow
  • 水平偏移量如果是正值向右偏移，负值向左
  • 垂直偏移量如果是正值向下偏移，负值向上
  • 偏移量和模糊度是相加计算的，偏移量的负号只是表示方向
  • 模糊度不能为负值
  • 偏移量和扩展也是相加的 box-shadow:5px 5px 5px 10px #fff;
  • 扩展和偏移量是数学运算
  • 边框阴影不会影响盒子布局

内阴影

  • box-shadow: inset 20px 20px 0
  • 关于模糊度，从一个颜色值在一定距离内进行一个渐变至透明的过程
  • 内外阴影：可以同时设置 box-shadow: inset 20px 20px 0 ，20px 20px 10px red；
  • 同时设置多个内外阴影： box-shadow: inset 20px 20px 0 ，inset 20px 20px 0 ，20px 20px 10px red，20px 20px 10px red；

背景

  • background-clip:content-box;可以修改背景区域
  • 默认背景填充区域就是border-box;
  • background-clip:padding-box;
  • background-origin:border-box;修改的是定位原点

  • background-origin: padding-box; 默认
  • background-size:100% 100%; 宽100%，高100%
  • background-size: auto;
  • background-size：contain;根据盒子的最小边进行缩放
  • background-size: cover;
  • cover会使“最大”边，进行缩放，另一边同比缩放，铺满容器，超出部分会溢出
  • 多重背景
    • background-image:url() no-repeat center top,url() no-repeat center bottom;

渐变（gradient）

  • 线性渐变
    • linear-gradient
    • 标准写法 background-image:line-gradient(yellow,green);至少有个颜色
    • 多个颜色渐变 background-image:line-gradient(yellow,green,red,blue);
    • 用角度确定方向，正上方为0度，顺时针旋转
      • background-image:line-gradient(90deg,yellow,green,red,blue);
    • 用关键字来确定方向
      • background-image:line-gradient(to top,yellow,green,red,blue);
      • background-image:line-gradient(to top left,yellow,green,red,blue);
    • 控制渐变
      • background-image:line-gradient(90deg,yellow 20%,green 40%);
  • 径向渐变
    • radial-gradient
    • background-image:radial-gradient(yellow,green);
    • 从一个中心点开始沿着四周方向进行渐变
    • background-image:radial-gradient(120px at center center,yellow,green);
      • 120px就是辐射的范围，center center就是原点
    • background-image:radial-gradient(120px at 80px 80px,yellow,green); 80px、80px中心位置的参照点是盒子的左上角
    • background-image:radial-gradient(120px 80px at 80px 80px,yellow,green); 辐射范围的半径可以不等即可以是椭圆 横轴，纵轴
    • background-image:radial-gradient(circle at 80px 80px,yellow,green); 圆形
    • background-image:radial-gradient(ellipse at 80px 80px,yellow,green); 椭圆
    • 辐射范围 中心点 颜色的起止
    • 用途：设置background-size：40px 40px;
  • 重复渐变
    • repeating-linear-gradient
    • background-image:repeating-linear-gradient(yellow 10%,green 40%)

过渡

  • transition

  • transition：all 1s

    • 当前元素只要有“属性”发生变化时，可以平滑的进行过渡

  • transition: left 2s;

    • transition-property:left;

    • transition-duration:2s;

    • transition-timing-function:linear;

      控制过渡的速度，匀速的

    • transition-timing-function:ease; 加速

    • transition-delay:5s;延时

    • 需要明确我要监听哪个属性的变化，并且规定变化的时间

过渡

  • 帧动画
  • 补间动画
    • 以一个或几个元素为主体，在一段时间内，将该主体各属性的一个值变成另一个值，我们只设置动画开始与结束状态时的属性
    • 动画——贝塞尔曲线
      • linear:匀速
      • ease:逐渐慢下来
      • ease-in：加速
      • ease-out：减速
      • ease-in-out：先加速后减速

动画

  • 相比过渡可以控制更多细节，可以将一个动画拆成多个步骤
  • 创建动画序列
    • @keyframes 动画序列名字{ 0%{} 100%{}}
  • animation-name:change;动画名称
  • animation-duration:1s;动画持续时间
  • animation-fill-mode:fillwards；动画结束后的状态
  • animation-iteration-count:infinite;无限次播放
  • animation-delay:1s；动画延时
  • animation-play-state:paused；动画暂停
  • animation-direction:alternate；动画结束后倒着播放回去
  • animation-timing-function:linear;
  • 有缩写形式：animation：move 1.5s infinite;

伸缩布局

  • 传统：需要精确计算宽度及外边距

  • 自动精确控制对齐，无需计算

  • 、、、css
    display:flex;
    justify-content: space-between;
    、、、

  • display:flex;伸缩布局

  • main axis主轴 cross axis侧轴(与主轴垂直)

  • flex-direction 可以调整主轴方向（默认水平方向）包括：row column row-reverse(从右向左) column-reverse

  • 怎么使用

    • 指定一个盒子为伸缩盒子
      • display：flex;
    • 设置属性来调整此盒的子元素的布局方式
      • flex-direction：column;
      • 它可互换主侧轴，也可改变方向
    • 明确方向
    • justify-content
      • 主轴方向对齐，可以调整元素在主轴方向上的对齐方式，包括flex-start flex-end center space-around（左右两边的距离一样） space-between（两边对齐）几种方式
      • 只能说开始和结束，因为主轴方向是可以改变的
    • align-items
      • 可以调整侧轴方向对齐方式，包括flex-start flex-end center baseline stretch (不指定高的情况拉伸)
    • flex-wrap
      • 控制是否换行，包括wrap，nowrap
      • 你让我不换行，我就忽略你的宽
    • align-content
      • 堆栈排列，可对应用flex-wrap:wrap后产生的换行进行控制，包括flex-start flex-end center space-between space-around stretch
    • flex-flow
      • 是flex-direction和flex-wrap的简写形式
      • flex-flow:column no-wrap;

  • 伸缩布局的子元素的属性

    • flex:1 控制子元素的伸缩比例，分配的是父盒子剩余的空间
    • align-self:同align-items可覆盖父元素设置的align-items，包括flex-start flex-end center baseline stretch
    • order：控制子元素的顺序 order：25

转换

  • 旋转、缩放、移动、倾斜、矩阵等
  • 旋转——rotate
    • 通过rotate（）方法，元素可以实现旋转效果，允许负值，元素将逆时针旋转。使用方法如：transform:rotate(45deg)/transform:rotate(-90deg)
  • 缩放——scale
    • 通过scale（）方法，可以更改元素的尺寸，可以分别调整x和y两个方向的大小。使用方法如：transform：scale（x,y) x或y可以是小数
  • 移动——translate
    • 通过translate（）方法，可以使元素从其当前位置移动，
